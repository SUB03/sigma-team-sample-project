# Purchase Service

Микросервис для управления покупками курсов.

## Функционал

- Создание покупки курса
- Получение списка покупок пользователя
- Получение ID купленных курсов
- Проверка, купил ли пользователь конкретный курс

## API Endpoints

- `POST /purchase/create/` - Создать покупку
- `GET /purchase/my-purchases/` - Список моих покупок
- `GET /purchase/my-courses/` - ID купленных курсов
- `GET /purchase/check/<course_id>/` - Проверить покупку
- `GET /purchase/all/` - Все покупки (админ)
